# CleanAir AI

CleanAir AI is a simple AI-powered system designed to predict short-term air quality and provide easy-to-understand health guidance.

The goal of this project is to make air pollution data more accessible and meaningful for everyday users.

## Motivation

Air quality data is often presented in complex charts and technical numbers. Many people struggle to interpret AQI values and understand how pollution affects their daily activities.

CleanAir AI aims to bridge this gap by combining basic machine learning predictions with human-friendly explanations.

## Features

- Short-term AQI prediction using historical data
- Simple health advisory based on predicted pollution levels
- Streamlit interface for easy interaction
- Lightweight and easy to expand

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
